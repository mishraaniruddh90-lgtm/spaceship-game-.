# 🚀 Space Runner 3D

A fast-paced **3D space runner game** built using **Three.js** where you control a spaceship, dodge incoming meteors, and survive as long as possible to achieve the highest score.

---

## 🎮 Gameplay

* Control a **spaceship** in space
* Avoid incoming **meteors ☄️**
* Score increases over time
* Game ends on collision

---

## 🕹️ Controls

| Key            | Action     |
| -------------- | ---------- |
| ⬅️ Left Arrow  | Move Left  |
| ➡️ Right Arrow | Move Right |

---

## ✨ Features

* 🚀 Realistic 3D Spaceship
* 🌌 Dynamic Space Background (Star Field)
* ☄️ Random Meteor Obstacles
* 📈 Live Score System
* 💥 Collision Detection
* 🎮 Smooth Controls & Camera Follow
* 🔥 Engine Flame Animation

---

## 🛠️ Built With

* JavaScript
* Three.js (3D Graphics Library)
* HTML5 & CSS3

---

## ▶️ How to Run

1. Download or clone the project
2. Open the folder in VS Code
3. Install Live Server extension
4. Right-click `index.html`
5. Click **"Open with Live Server"**
